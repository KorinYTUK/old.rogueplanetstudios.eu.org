ESPLCompiler Documentation

The ESPLCompiler is a program that compiles ESPL (HTML But Simpler!) code into HTML code. It processes a set of directives defined in the ESPL code and generates the corresponding HTML output. Directives The ESPLCompiler recognizes the following directives:
  • title: Defines the title of the HTML document.
  • h1: Creates an

    heading.

  • h2: Creates an

    heading.

  • p: Creates a paragraph

    .

  • link: Creates an anchor link.
Usage The ESPLCompiler can be used as a command-line tool with the following options: espl -build -o -build, -b: Specifies the input ESPL file to compile. -o, --output: (Optional) Specifies the output HTML file. The default output file is index.html.
